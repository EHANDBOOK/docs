Supported Input File Formats and Versions

EHANDBOOK Container-Build supports various input file formats.

The following table provides an overview on the supported versions of the specific file formats as well as where to find more details.

  • Other versions of the file formats may also work as well, but are not tested and supported by ETAS.

  • For several formats, a subset of the functionality is supported by EHANDBOOK. Please refer to the EHANDBOOK-specific specifications.
