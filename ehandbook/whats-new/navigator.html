What’s New

Version 14.1 (March 2026)

Importing comments to new containers

  • Comments can be added to EHANDBOOK containers to enrich the documentation.

  • Comments can be imported to other containers. The import ensures that no comment gets lost, even though the containers might have changed.

Version 14.0 (January 2026)

Model Comments

  • Enrich interactive models and function overviews by adding comments.

  • Easily export and transfer comments to other containers.

Filter Compare results by type

  • Focus on relevant changes by filtering the results.

  • Decide whether to look at text differences, table differences or image differences.

Interoperability with Vision

  • Connect to a running Vision instance and display live measurement values in your EHANDBOOK interactive model.

  • Supports a read-only mode as well as a mode that helps set up a Vision project.

Version 13.5 (November 2025)

Improvement: Ignore Whitespaces in Compare

  • Reduce the effort by ignoring changes that only affect whitespaces.

  • Easily focus on functional changes between two container versions.

New feature: Save Model Views

  • Save customized model views to easily open them again.

  • Transfer your customized model views to other containers.

Version 13.4 (November 2025)

New feature: Export labels to LAB file

  • Easily transfer information for relevant labels to other tools.

New feature: Search in Compare results

  • Find relevant information within the compared content.

Improvement: Automatic detection of A2L encodings

  • Choose the right encoding to correctly load you A2L data manually or automatically.

  • Supports also A2L files which contain parts in different encodings.

Improvement: Restoration of open tabs

  • Automatically restore your open tabs after a tool restart to easily continue your work.

  • Zooming, Scroll position, applied layouts, expanded blocks and applied compact view are restored to the previous state.

Version 13.3 (September 2025)

Improvement: Show detailed image differences

  • Compare images in two containers in detail.

  • Detect even small modifications between the images.

Version 13.2 (June 2025)

Improvement: Restore open tabs after restart

  • Continue your work even after EHANDBOOK-NAVIGATOR has been closed and restarted.

  • After the tool is restarted, all documentation, model and function overview tabs that were previously opened will be restored.

Version 13.1 (April 2025)

Improvement: Filter the Compare Tree

  • Easily filter the comparison results by name.

  • Custom filtering by using wildcards.

Version 13.0 (March 2025)

Improvement: Distinguish Modified Entries

  • Easier identification of modified entries in the Compare results

  • Focus on modifications only

Improvement: Label Popup and Swapping Containers

  • Label Popup is available also in Compare Viewers

  • Compared containers can be easily swapped

Improvement: Synchronize Cursor positions with CANape

  • Transfer cursor positions from CANape to EHANDBOOK-NAVIGATOR in an interactive mode.

  • Benefit from viewing the values within the interactive model in EHANDBOOK-NAVIGATOR.

Version 12.4 (December 2024)

Improvement: Use external containers for comparison

  • Compare containers within your EHANDBOK project with externally located containers.

  • Containers can be chosen from your file system without modifying your EHANDBOOK project.

Improvement: Image comparison

  • Compare images within your documentation.

  • Detect and explore differences.

New feature: ESDL code viewer

  • Include ESDL code into your documentation.

  • The ESDL code can be opened within EHANDBOOK and is included into search.

Version 12.3 (October 2024)

Improvement: Hide C-Code functions based on function name

  • C-Code functions can be specified as non-expandable for interactive models in EHANDBOOK-NAVIGATOR during EHANDBOOK Container generation.

  • A JSON file specifies the list of functions to be hidden.

New feature: Compare

  • New add-on that allows to compare two arbitrary containers.

  • Explore and review the differences using a side-by-side viewer with synchronized scrolling.

Version 12.2 (September 2024)

Improvement: Improved Navigation History

  • Navigation steps are recorded and can easily be retraced using the navigation history.

  • It is available for documentation content and interactive models.

Version 12.1 (July 2024)

New feature: Add and remove attachments

  • Arbitrary files can now be added to the documentation container as attachments.

  • Removing an attachment is also possible.

  • Changes will only be made to the documentation container when saved.

Improvement: Support for Simulink Truth Table block

  • Truth Table is a block in Matlab that implements the design of combinational logic in a tabular format.

  • Ehandbook now supports these truth table block.

Improvement: Support for Simulink Variant blocks

  • Variant source block, the active choice appears normally and the inactive choice appears faded.

  • EHANDBOOK now supports processing these variant choices.

Improvement: No Value Information in Offline Measurement Data

  • Correctly display "n/a" when no value information is available for a label at a certain time

Version 12.0 (April 2024)

Improvement: Bookmark icon in the table of contents and the model explorer

  • Bookmarked entries are now marked by a blue icon to easier identify the relevant locations.

New feature: Interoperability with CANape - showing live values

  • Support users while performing live tests

  • Show live measurement values within the interactive model while connected to a CANape application

New feature: MATLAB Function Block Code Viewer

  • EHANDBOOK-NAVIGATOR 12.0 introduces a code viewer that can display code written within MATLAB function blocks.

Version 11.3 (December 2023)

Improvement: Raster Pre-Selection and Raster allocation in case of overflow

When connecting EHANDBOOK-NAVIGATOR to INCA you can influence the raster allocation by:

  • Overwriting the default raster using raster pre-selection, e.g. in case no default raster has been defined in the A2L file.

  • Specifying a list of rasters to use in case a raster cannot be assigned to its default raster.

Improvement: Sorting and Filtering in Shortcut Overview

  • Find shortcuts easier by sorting alphabetically.

  • Filter the list of shortcuts by entering arbitrary search terms.

Improvement: Updated Look and Feel

  • Improved look and feel of EHANDBOOK-NAVIGATOR by adopting the ETAS UI theme.

Version 11.2 (September 2023)

Improvement: Copy calibration data from a table

  • Calibration data can be useful in other contexts

  • Copy the table to the clipboard right from the label popup

New feature: Modify, share and open location

  • Modify and copy location directly from the navigation bar

  • Easy sharing of locations

  • Open a new editor with a specific content

Version 11.1 (June 2023)

New feature: PDF Support

  • Integrate PDF content into your EHANDBOOK Container.

  • Display PDF content seamlessly in EHANDBOOK-NAVIGATOR.

  • Use selection and copy as usual and include PDF content into your search.

Improvement: Importing Bookmarks From Previous Versions

  • Easily transfer bookmarks from older versions on first tool start.

  • Keep Bookmarks on uninstallation.

Improvement: A2L Encoding Selection

  • EHANDBOOK-NAVIGATOR proposes matching encodings, ordered by match quality.

  • Manually change the encoding, e.g. when using special characters.

Improvement: Number Formatting Options

  • Configure display of values (number of decimal places, trailing zeros).

  • Decide whether to use externally given formats.

Version 11.0 (March 2023)

Improvement: Enhanced Bookmarking Feature

  • Easily memorize places in the ECU software documentation.

  • Create bookmarks for places in textual documentation and interactive models directly from the content.

  • Use bookmarks with different EHANDBOOK Container files.

Version 10.3 (December 2022)

New feature: Model Navigation Bar

  • Keep orientation while exploring large interactive models.

  • Navigate between model levels more quickly.

Improvement: Configure which cursor to display when annotating measurement values

  • Configure to show values for cursor A, cursor B, or both.

  • Obtain an annotated interactive model which is less cluttered.

Version 10.2 (September 2022)

Improvement: Show/hide individual interface elements

  • Only show relevant interface elements for AUTOSAR (Classic) ports.

  • Obtain overview diagrams that are less cluttered and more understandable.

Improvement: Advanced search criteria for finding information in the current tab

  • New option to precisely find case-sensitive text.

  • Advanced option to find information in deeply nested interactive models.

Improvement: Configure colors for highlighting search hits

  • Choose a custom color for highlighting of search hits.

  • Spot highlighted search hits more easily and quickly.

Version 10.1 (June 2022)

Improvement: Improved representation of binary values in label popup

  • Find individual bits in bit mask labels faster.

  • Read and grasp status of a bit more efficiently.

Improvement: Performance Improvement Search in Interactive Models

  • Search and navigate in large interactive models much smoother.

  • Find what you are looking for faster.

Improvement: Support for additional AUTOSAR Classic port interface types

  • Obtain information on different interface types and their elements through search proposals and label popup.

  • Understand connections between software components based on different port interface types.

Version 10.0 (March 2022)

New feature: Attachments

  • Obtain additional information through file-based attachments to EHANDBOOK Containers.

  • Opening and saving attachments.

Improvement: Enhanced support for overview diagrams derived from AUTOSAR (Classic)

  • Show interface elements within ports directly in overview diagrams.

  • Annotate values from connected INCA experiment or a loaded measurement file to interface elements.
