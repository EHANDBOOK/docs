Menus

The EHANDBOOK-NAVIGATOR contains the following menus:

File Menu

The File menu contains the following menu items:

  • Open…​

    • Opens a new container.

  • Convert A2L file…​

    • Helps to convert A2L file to EHANDBOOK container.

  • Save…​

    • Saves the changes done in the current project or the container.

  • Save As…​

    • Helps to save a copy of the currently opened project or the container with the new name.

  • Manage Project…​

    • Helps to load one or more containers into an EHANDBOOK project.

  • Print…​

    • Displays the Print dialog box. In the dialog box, specify the printer and invoke the printing action.

  • Extract Content…​

    • Helps to extract content from the currently opened container to a new EHANDBOOK container.

  • Export to PDF…​

    • Exports the contents of a functional component to a PDF file.

  • Import

    • Imports bookmarks to the Bookmarks view and imports comments from one container to another container.

  • Export

    • Exports bookmarks from Bookmarks view.

  • Preferences

    • Displays the Preferences dialog box which helps to set the options to operate the EHANDBOOK-NAVIGATOR.

  • List of previously loaded containers

    • This helps you to conveniently switch to other containers.

  • Close

    • The EHANDBOOK-NAVIGATOR tool is sent to the Windows tray instead of exiting.

  • Exit

    • Quits the EHANDBOOK-NAVIGATOR.

Search Menu

The Search menu contains the following menu items:

  • Find in EHANDBOOK File(s)

    • Find in EHANDBOOK File(s) is activated. The cursor is placed in the Find in EHANDBOOK File(s) field.

  • Find in current tab

    • Find in current tab is initiated. The cursor is placed in the Find in current tab field.

View Menu

The View menu contains the following menu items:

  • Hide Toolbar

    • Hides the toolbar.

  • Show Toolbar

    • Restores the toolbar to its original position. This menu item is displayed in the Window menu in place of Hide Toolbar, only if the toolbar is hidden.

  • Show View

    • Allows you to activate or restore the following views, if they are closed:

      • Bookmarks

      • Document Explorer

      • Model Explorer

      • Search Results

      • Outline

      • Properties

  • Content Perspective

    • Switches to Content Perspective.

  • Extended Perspective

    • Switches to Extended Perspective.

  • Personal Perspective

    • Switches to Personal Perspective.

  • Reset Perspective

    • Restores the default views of the EHANDBOOK-NAVIGATOR to their original positions.

  • Full Screen Mode

    • Resizes the EHANDBOOK-NAVIGATOR to the entire screen.

  • Zoom out

    • Decreases the size of the contents.

  • Zoom in

    • Increases the size of the contents.

  • Restore Default Zoom (100%)

    • Restores the contents in 100% size.

  • Fit To View

    • Adjusts the zoom level in such a way that all the contents in the selected Model Viewer fit into the page and aligned centrally.

The below menus will be visible in the View menu only when Function overview diagrams or Interactive models are opened:

  • Expand

    • Expands the selected block and displays the relevant hierarchy level. This menu is enabled only for expandable blocks.

  • Collapse

    • Collapses an expanded hierarchy level. This menu is enabled only for expanded blocks.

  • Normal View

    • Applies normal view to a function overview diagram.

  • Compact View

    • Applies compact view to a function overview diagram.

  • Show Interface Elements

    • Displays interface elements of relevant ports of function overview diagram.

  • Hide Interface Elements

    • Hides interface elements of relevant ports of function overview diagram.

  • Default Layout

    • Generates the model layout as designed in the authoring tool (ASCET, Simulink, and C-Code based models).

  • Auto Layout

    • Generates the model layout optimized by the built-in layout algorithms of EHANDBOOK-NAVIGATOR to the interactive models of ASCET, Simulink models and Function Overview Diagram only. Auto layout is not applicable to C-Code based models.

  • Remove Highlighting

    • Removes the signal flow highlighting/hiding from the model.

Data Menu

The Data menu contains the following menu items:

  • INCA Experiment (Live)

    • Connects the EHANDBOOK-NAVIGATOR to a running INCA experiment.

  • Measurement and Calibration Files

    • Allows the visualization of previously recorded values and connects from MDA 8 for measuring data.

  • None

    • None of the external applications are connected. By default, this is selected.

  • Link EHANDBOOK Files to Hardware Devices

    • Links EHANDBOOK files to hardware devices to visualise the measurement data that is captured in those particular devices.

  • Display all Values

    • Displays all the values which are generated from the connected INCA experiment and Measurement File in the interactive model shown in the Model Viewer. In INCA, it marks all labels to show values from the experiment. In Measurement Files, it also marks all labels to show the recorded values. No values will be shown for unknown labels.

  • Hide all Values

    • Hides all the values which are generated from the connected INCA experiment and Measurement File in the interactive model shown in the Model Viewer.

  • Start Visualization

    • Starts the visualization in the connected INCA experiment and displays live values for measurement variables and calibration parameters in the Model Viewer.

  • Stop Measuring

    • Stops the running measurements in the connected INCA experiment.

Help Menu

The Help menu contains the following menu items:

  • Welcome

    • Displays the EHANDBOOK-NAVIGATOR welcome page.

  • Keyboard Shortcuts…​

    • Displays all the keyboard shortcut keys provided by EHANDBOOK-NAVIGATOR.

  • Documentation

    • Displays the EHANDBOOK-NAVIGATOR documenation.

  • Support…​

    • Provides EHANDBOOK-NAVIGATOR support information.

  • Export Log Files…​

    • Exports the necessary log files into a zip file.

  • ETAS Safety Advice

    • Displays the safety advice for EHANDBOOK-NAVIGATOR.

  • ETAS License Manager

    • Displays the ETAS Licensing Manager dialog box. The dialog box allows you to install, manage, and check the status of your licenses.

  • About EHANDBOOK-NAVIGATOR

    • Provides information and license details for the components, third party and open source software used in EHANDBOOK-NAVIGATOR.
