Toolbars

Main Toolbar

EHANDBOOK-NAVIGATOR Toolbar is a strip of icons which helps you select the EHANDBOOK-NAVIGATOR operations easily. Toolbar comprises of the following buttons:

Opens a new container

Saves the changes made in the container

Displays the Manage EHANDBOOK Project dialog box

Opens the Print dialog box

Opens the Extract content to a new EHANDBOOK container window

Opens the Export to PDF window

Navigates to the previous topic

Navigates to the next topic

Search field to enter the tex

Search options while searching the entered text

Displays the contents of previous search hit

Displays the contents of next search hit

Helps to create a new function overview diagram

Applies normal view to a function overview diagram

Applies compact view to a function overview diagram

Displays interface elements of relevant ports of function overview diagram

Hides interface elements of relevant ports of function overview diagram

Applies default layout to the interactive model

Applies the auto layout mode to the interactive model

Removes highlighting/hiding from the model

Clips and displays only the contents of the selected block

Navigates into the contents inside the selected block diagram. When navigating in from a hierarchy level, the previously visited layout will be retained. If it is visited first time, then the Default layout is shown.

Navigates out from the model to its previous level. Helps to navigate out to view its Function Overview diagram. When navigating out from the parent layout, always the Default layout is shown

Displays the content of parent hierarchy while also keeping the current content intact

Export the model/function overview diagram as an image

Displays all the keyboard shortcut keys provided by EHANDBOOK-NAVIGATOR

Displays the EHANDBOOK-NAVIGATOR documentation in a new tab

Displays the EHANDBOOK-NAVIGATOR welcome page in a new tab, which can be closed if not required

Opens the model or documentation location

Bookmarks View toolbar comprises of the following buttons:

Import Bookmarks

Export Bookmarks

Maximize the view

Minimize the view

Status Bar

EHANDBOOK-NAVIGATOR status bar is similar to toolbar which comprises of icons. It also helps you to select the EHANDBOOK-NAVIGATOR operations easily. Status Bar comprises of the following buttons:

Opens the ETAS Internet site for more details on EHANDBOOK-NAVIGATOR and its license details

Drop-down menu to select other ETAS Products such as INCA Experiment or MCD Devices

Links EHANDBOOK files to individual hardware devices in order to define the mapping of the functions to ECU

Displays the Document Viewer/Model filled with available devices of the measurement file Viewer in full screen

Restores the default views of the EHANDBOOK-NAVIGATOR to their original positions

Displays the views of the EHANDBOOK-NAVIGATOR in the positions you have personalized

Increases the size of the contents

Decreases the size of the contents

Displays the contents in 100% size

Adjusts the zoom level in such a way that all the contents in the selected Model Viewer fit into the page and aligned centrally

Displays all measurement and calibration values

Hides all measurement and calibration values of the EHANDBOOK-NAVIGATOR in the positions you have personalized

Displays all values for the highlighted signals

Starts running the INCA experiment and displays the values for measurement variables and calibration parameters

Stops the INCA experiment

Reset Perspective

Reset Perspective feature helps to restore all views of the EHANDBOOK- NAVIGATOR to their original/default positions.

To restore the views of the EHANDBOOK-NAVIGATOR to their default position:

  • Go to View menu and select Reset Perspective ⇒ Reset Perspective dialog box is displayed.

  • Click Yes to restore all views to their default positions. Document Explorer, Document Viewer, Model Explorer, Outline View, Properties View, and Bookmarks View are restored to its default position.

The zooming is available on the View menu or the Status bar in the Document Viewer/Model Viewer.
