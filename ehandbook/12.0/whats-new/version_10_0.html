Version 10.0 (March 2022)

NAVIGATOR

Attachments

  • Obtain additional information through file-based attachments to EHANDBOOK Containers.

  • Opening and saving attachments.

Enhanced support for overview diagrams derived from AUTOSAR (Classic)

  • Show interface elements within ports directly in overview diagrams.

  • Annotate values from connected INCA experiment or a loaded measurement file to interface elements.

Container-Build

  • Improvement: Adding attachments to EHANDBOOK containers

  • Improvement: Default behavior for mask generation has changed to Simulink Mask Decorations as Image. The earlier option -DenableSimulinkMaskDecorationsAsImage is deprecated.

  • Linux products available for EHB-CB and UGG

  • The AUTOSAR (Classic) addon supports the Simulink AUTOSAR blockset
